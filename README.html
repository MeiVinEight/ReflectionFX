ReflectionFX

ReflectionFX是一个全平台支持的，高效易用的反射库
ReflectionFX的设计初衷在于高效率，在此基础上以易用为目的开发

介绍

  • ReflectionFX 是一个以效率为主的高效反射库
  • ReflectionFX 可以访问到任何存在于运行时的元素
  • ReflectionFX 没有强制的异常捕获要求
  • ReflectionFX 支持了泛型返回值，省去了开发时接收返回值的强制转换
经测试
ReflectionFX 的调用效率远超 java.lang.reflect.Method#invoke(Object, Object...)
与 java.lang.invoke.MethodHandle#invoke(Object...) 持平，接近直接调用

测试结果参考:
类型 测试次数 平均用时 误差 单位
Method 104 6094.443 62.931 ns/op
MethodHandle 104 249.151 3.348 ns/op
ReflectionFactory 104 249.778 3.381 ns/op
PolymorphismFactory 104 250.766 2.271 ns/op
MagicAccessFactory 104 248.562 1.162 ns/op
直接调用 104 248.705 1.591 ns/op
  • ReflectionFactory PolymorphismFactory MagicAccessFactory 为 ReflectionFX 支持的反射类型

开始

ReflectionFX 支持三种类型的调用方式

  • ReflectionFactory
    ReflectionFactory 和 MethodHandle 类似，其中提供了对方法，字段，构造方法的调用获取
  • PolymorphismFactory
  • MagicAccessFactory
